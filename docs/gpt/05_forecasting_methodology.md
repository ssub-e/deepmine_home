# Forecasting Methodology

## Forecasting Philosophy

Forecasting is the process of estimating future outcomes using historical data and external variables.

Forecasting differs from:

- Goals
- Plans

Forecasting should represent the most likely future outcome.

---

## Forecasting Workflow

1. Define business problem
2. Collect data
3. Clean data
4. Feature engineering
5. Model selection
6. Model training
7. Validation
8. Deployment
9. Monitoring
