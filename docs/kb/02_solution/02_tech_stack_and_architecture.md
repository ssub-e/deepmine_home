# 기술 스택 및 AI 아키텍처

## 1. AI 프레임워크 기술 스택

딥마인 솔루션은 높은 확장성과 재현성을 담보하기 위해 검증된 오픈소스 라이브러리와 독자적인 최적화 엔진을 긴밀하게 연계하여 구동합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                      Exogenous Context Data                         │
└────────────────────────────────┬────────────────────────────────────┘
                                 ▼
┌─────────────────────────────────────────────────────────────────────┐
│           Data Augmentation (Tsaug, SMOTE) & Preprocess             │
└────────────────────────────────┬────────────────────────────────────┘
                                 ▼
┌─────────────────────────────────────────────────────────────────────┐
│  Predictive Engines (DeepAR, TFT, Foundation Models, Tree-based)   │
└────────────────────────────────┬────────────────────────────────────┘
                                 ▼
┌─────────────────────────────────────────────────────────────────────┐
│ Optimization (Bayesian, Optuna, Ray Tune) & Loss (MAE/MSE)         │
└─────────────────────────────────────────────────────────────────────┘
```

### 1-1. 머신러닝/딥러닝 프레임워크

| 프레임워크 | 역할 |
|-----------|------|
| **Apache MXNet** | 빠른 하드웨어 연산 가속 및 분산 학습 지원 |
| **PyTorch** | 모델 커스터마이징 및 최신 딥러닝 알고리즘의 유연한 연구 개발 보장 |

### 1-2. 예측 알고리즘 범주

| 알고리즘 | 유형 | 설명 |
|---------|------|------|
| **DeepAR** | 딥러닝 (RNN) | 확률적 자가회귀 순환 신경망 모델 |
| **TFT** | 딥러닝 (Transformer) | 시간적 가변성과 정적 변수를 아우르는 트랜스포머 기반 모델 |
| **XGBoost / LightGBM** | 머신러닝 (Tree) | 정형 시계열에 강한 트리 앙상블 계열 |
| **Foundation Models** | 파운데이션 | 대형 데이터로 사전 학습된 제로샷 시계열 기반 예측 아키텍처 |

### 1-3. 하이퍼파라미터 최적화 (HPO) 알고리즘

| 알고리즘 | 설명 |
|---------|------|
| **Bayesian Optimization / Optuna** | 베이지안 사후 분포 기반 탐색 영역 자동 조율 |
| **AutoNHITS / AutoMLP** | 최적의 노드 수와 입력 윈도우 크기를 자동으로 추정 |
| **Hyperband** | 자원의 조기 종료(Pruning) 기법을 결합하여 탐색 속도 극대화 |

### 1-4. 데이터 증강 및 전처리 알고리즘

| 기법 | 설명 |
|------|------|
| **Tsaug** | 시계열 데이터 전용 드리프트, 스케일링, 노이즈 삽입을 통한 데이터 왜곡 보강 |
| **SMOTE** | 불균형한 이상 데이터 분포 및 급격한 스파이크 보정 기법 |
| **Tsfresh / tslearn** | 자율적인 대규모 피처 추출(Feature Extraction) 및 시계열 군집화 분석 지원 |

## 2. AI 예측 아키텍처 모델군 분류

본 솔루션은 시계열 특성과 연산 요구 사양에 맞춰 **4대 모델군** 중 최적의 성능을 발휘하는 구조를 동적으로 선택합니다.

```
               ┌───────────────────────┐
               │   AI 아키텍처 모델군   │
               └───────────┬───────────┘
                           │
    ┌──────────────┬───────┴──────┬──────────────┐
    ▼              ▼              ▼              ▼
┌───────┐      ┌───────┐   ┌───────────┐   ┌───────┐
│  MLP  │      │  RNN  │   │Transformer│   │  CNN  │
│Family │      │Family │   │  Family   │   │Family │
└───────┘      └───────┘   └───────────┘   └───────┘
```

### 2-1. MLP Family

**대표 모델**: NHITS, NBEATS, MLP

| 항목 | 설명 |
|------|------|
| **예측 방식** | Direct Forecasting — 단일 학습 프로세스 내에서 모든 미래 예측 시점을 한 번에 산출 |
| **핵심 강점** | 중간 시점들의 오차가 누적 전파되는 오류 누적(Error Propagation) 현상이 없으며, 병렬 연산 속도가 매우 빠름 |
| **적합 시나리오** | 다수의 SKU를 대량 일괄 예측해야 하는 대규모 수요 예측 |

### 2-2. RNN Family

**대표 모델**: LSTM, GRU, DeepAR

| 항목 | 설명 |
|------|------|
| **예측 방식** | Recursive Forecasting — 다음 시점의 값을 예측하여 이를 다시 입력으로 취해 그 다음 시점을 예측 |
| **핵심 강점** | 시계열의 순차적(Sequential) 흐름과 장기 이력을 유연하게 수용하여 가변적인 이력 분석에 적합 |
| **적합 시나리오** | 이력 길이가 가변적이거나, 확률적 예측 분포가 필요한 경우 (DeepAR) |

### 2-3. Transformer Family

**대표 모델**: Autoformer, TFT (Temporal Fusion Transformer), PatchTST

| 항목 | 설명 |
|------|------|
| **예측 방식** | Multi-Head Self-Attention — 입력의 특정 시간 영역에 가중치를 집중 |
| **핵심 강점** | 장기 의존성(Long-term dependency) 학습 능력이 독보적이며, 동적 상황 분석 및 외생 변수 융합 성능이 강함 |
| **적합 시나리오** | 외생 변수(환율, 프로모션 등)를 적극 활용해야 하는 복합 예측, 장기 예측 |

### 2-4. CNN Family

**대표 모델**: BiTCN, Temporal Convolutional Filters

| 항목 | 설명 |
|------|------|
| **예측 방식** | 시간 도메인을 따라 팽창 합성곱(Dilated Convolution) 필터를 밀어내며 연산 |
| **핵심 강점** | 수용 영역(Receptive Field)을 넓게 확보하면서 국소 시계열 패턴을 고속으로 탐색하는 데 특화 |
| **적합 시나리오** | 주기적 패턴이 명확한 데이터, 실시간 추론 속도가 중요한 경우 |

## 3. 모델군 비교 요약

| 모델군 | 예측 방식 | 속도 | 장기 의존성 | 외생 변수 활용 | 대표 모델 |
|--------|----------|------|------------|--------------|----------|
| **MLP** | Direct | ⚡ 매우 빠름 | 보통 | 보통 | NHITS, NBEATS |
| **RNN** | Recursive | 보통 | 좋음 | 좋음 | DeepAR, LSTM |
| **Transformer** | Attention | 보통 | 우수 | 우수 | TFT, PatchTST |
| **CNN** | Convolution | 빠름 | 좋음 | 보통 | BiTCN |
