# TimeGPT

## Overview

TimeGPT is a transformer-based foundation forecasting model.

## Architecture

Encoder-Decoder Transformer

Key Features:

- Self-attention
- Positional encoding
- Residual connections

## Training Scale

More than 100 billion data points.

Domains:

- Finance
- Energy
- Weather
- IoT

## Usage

### Zero-Shot

Immediate forecasting.

### Fine-Tuning

Domain adaptation.

### Exogenous Variables

Supports external variables.
